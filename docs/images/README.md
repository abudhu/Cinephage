# Images

This folder contains screenshots and images for Cinephage documentation.

## Naming Convention

Use descriptive, lowercase names with hyphens:

- `dashboard-overview.png` - Main dashboard view
- `settings-indexers.png` - Indexer configuration page
- `search-results.png` - Search results example
- `library-movies.png` - Movie library view

## Recommended Specs

- **Format**: PNG for screenshots, SVG for diagrams
- **Width**: 800-1200px for full-screen captures
- **Quality**: Optimize file size while maintaining clarity

## Usage in Docs

Reference images using relative paths:

```markdown
![Dashboard Overview](images/dashboard-overview.png)
```
